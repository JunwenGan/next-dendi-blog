## Introduction

As a Frontend Developer in 2025, my development workflow is centered around tools that enable me to work faster and more effectively. The right tools help me write clean code, manage projects efficiently, and maintain a highly optimized environment.

## My Stack

Here's what I use daily:

### Code Editor

**VS Code** remains my primary editor, enhanced with these extensions:
- GitHub Copilot for AI-assisted coding
- Prettier for code formatting
- ESLint for catching issues early
- GitLens for git superpowers

### Framework

**Next.js 15** is my go-to framework:
- App Router for modern React patterns
- Server Components for performance
- Built-in optimization features

<Callout type="tip" title="Pro Tip">
Use `turbopack` in development for significantly faster hot module replacement. Just add `--turbopack` to your dev script.
</Callout>

### Styling

**Tailwind CSS** + **shadcn/ui** is the perfect combo:
- Utility-first approach speeds up development
- Consistent design system
- Great dark mode support

### Database & Backend

- **Prisma** for type-safe database access
- **PostgreSQL** (via Neon) for reliable data storage
- **NextAuth** for authentication

## Productivity & Utilities

- **Warp** - Modern terminal with AI features
- **Raycast** - Launcher on steroids
- **Arc Browser** - My daily driver for web browsing
