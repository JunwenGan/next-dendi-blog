## Overview

GourmetVision is an AI-powered application that transforms restaurant menus into visual experiences. Simply upload a menu image, and the app will extract dish information and generate realistic food photos for each item.

## Key Features

### Menu Scanning & Analysis
Using Google Gemini's vision capabilities, the app extracts dish names, descriptions, and prices from uploaded menu images. The AI understands various menu formats and layouts.

### AI Food Photo Generation
For each extracted dish, OpenAI generates a realistic, appetizing food photo. The generation considers the dish description to create accurate visual representations.

### Multi-Language Support
The app automatically detects and translates menu items to English, making it perfect for travelers exploring foreign restaurants.

### Interactive Gallery
Browse all generated food photos in a beautiful gallery view. Each photo includes the dish name, description, and price.

## Technical Stack

- **Frontend**: Next.js with TypeScript for type-safe development
- **Styling**: Tailwind CSS for responsive, modern UI
- **AI Integration**:
  - Google Gemini for menu analysis and text extraction
  - OpenAI for food photo generation
- **Database**: Prisma ORM for storing menu data

## Use Cases

- Travelers wanting to visualize dishes at foreign restaurants
- Restaurants looking to create visual menus quickly
- Food bloggers generating content ideas
- Anyone curious about what a dish looks like before ordering
