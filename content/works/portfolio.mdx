## Overview

This portfolio website showcases my work as a full-stack developer. Built with modern web technologies, it features a custom design system, smooth animations, and a content management system powered by MDX.

## Key Features

### Bento Grid Layout
The homepage features a unique bento grid design with interactive elements including:
- Interactive 3D globe showing timezone flexibility
- Technology stack scroller
- Project marquee showcasing current work

### Blog System
A full-featured blog built with MDX, supporting:
- Rich markdown content with custom components
- Syntax highlighting for code blocks
- Table of contents with scroll tracking
- Tag-based filtering and search

### Work Showcase
Portfolio section displaying projects with:
- Project cards with tech stack badges
- Detailed project pages with MDX content
- GitHub and live demo links

### Animations
Smooth, performant animations throughout using Framer Motion:
- Scroll-based skill section animations
- Page transitions
- Hover effects and micro-interactions

## Technical Implementation

### Frontend
- **Next.js 15** with App Router for server-side rendering
- **TypeScript** for type safety
- **Tailwind CSS** for styling with custom design tokens
- **Framer Motion** for animations

### Content Management
- **MDX** for blog posts and project pages
- Custom components for callouts, code blocks
- Automatic TOC generation from headings

### Backend
- **Prisma ORM** with PostgreSQL for dynamic data
- **NextAuth.js** for authentication
- **Neon** for serverless PostgreSQL hosting

### Deployment
- **Vercel** for hosting and CI/CD
- Automatic deployments on git push
